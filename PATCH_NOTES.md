# WorldWar Dashboard — Phase 3 Patch

This patch adds the Phase 3 “defensibility layer”:

- Lens presets (Editorial / Armed Conflict / War-Connectedness / Coercion)
- Evidence section in the info panel (collapsed by default)
- About/Methodology modal (caveats hidden behind an intentional disclosure)

## Files in this patch

- `assets/app.js`
- `data/schema.json`
- `data/map_data.json` (only metadata changed vs the starter; nodes/edges unchanged)

If you already edited your `data/map_data.json`, **do not overwrite it blindly** — instead copy only the `metadata.about` block from this patch.
